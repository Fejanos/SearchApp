Projekt: DFS és BFS kereső labirintusban – grafikus vizualizációval

Ez a projekt oktatási célra készült, és a Harvard CS50's Introduction to AI with Python című kurzusának "Search" témakörén alapul.

Az eredeti algoritmusötlet a CS50 előadásból származik, de a megvalósítás, a grafikus kezelőfelület (pygame),
valamint a kódstruktúra kisebb-nagyobb átdolgozáson esett át saját fejlesztés keretében.

Forrás: https://cs50.harvard.edu/ai/2024/
-----------------------------------------------------------------------------------------------------------------------

Fejlesztési környezet – VS Code & Python
1. VS Code letöltése: https://code.visualstudio.com/
2. Python telepítése: https://www.python.org/downloads/
3. VS Code bővítmény telepítése: Python (a Microsoft-féle)
4. Interpreter kiválasztása:
   View → Command Palette → Python: Select Interpreter → válaszd ki a megfelelő python.exe-t
-----------------------------------------------------------------------------------------------------------------------

Külső csomag telepítése (pygame)
1. Nyisd meg a terminált
2. Futtasd:

   pip install pygame

   vagy (ha több interpreter van):

   python -m pip install pygame
